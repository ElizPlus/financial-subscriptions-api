# Financial Subscriptions API

RESTful API для управления финансовыми подписками пользователя, разработанная на Flask.

## Функциональность

- Создание подписок
- Просмотр всех активных подписок
- Редактирование подписок
- Удаление подписок (мягкое удаление)
- Логирование действий в аудит-лог
- Интеграция с PostgreSQL
